# A PyQt5 Mesh Viewer

## Dependencies

* pyqt5
* moderngl
* pyrr
* openmesh

## Usage

1. Run
    ```
    python ./mgl_qt_meshviewer.py
    ```
2. Click ```File > Open``` in the menu bar to select a mesh file.
3. Drag the left mouse button to rotate.
